## 1. 제안 개요

### 1.1 무엇을 제안하는가

> **AI 개발 조직이 기술력을 신뢰 자산으로 전환하는 자기확장형 콘텐츠 플랫폼**

바이칼시스템즈는 AI 솔루션 개발 과정에서 축적된 기술 인사이트를 외부에 체계적으로 공유하고,  
이를 SEO 자산·브랜드 자산·영업 자산으로 전환하는 플랫폼 인프라를 구축했습니다.

이 플랫폼은 바이칼시스템즈 자체 운영에 그치지 않고,  
**동일한 문제를 가진 AI 스타트업 또는 기술 조직에 화이트라벨 또는 SaaS 형태로 제공**할 수 있는 확장성을 갖추고 있습니다.

## 2. 시장 배경 및 문제

### 2.1 AI 스타트업의 공통 고민

2026년 현재 국내외 수많은 AI 스타트업이 기술력을 보유하고 있으나,
**기술력을 신뢰로 전환하지 못하는 구조적 문제**를 공통으로 안고 있습니다.

| 증상 | 원인 |
|------|------|
| 홈페이지 방문자 수 저조 | 방문 이유가 없는 정적 소개 페이지 |
| 영업 자료 부족 | 기술 성과가 외부에 보이지 않음 |
| SEO 트래픽 0 | 콘텐츠가 외부 플랫폼에 분산·소멸 |
| 채용 지원자 부족 | 기술 문화가 외부에 전달되지 않음 |

### 2.2 기존 해결 방식의 한계

| 기존 방식 | 문제점 |
|-----------|--------|
| 네이버 블로그 | SEO 자산이 네이버에 귀속, 콘텐츠 분산 |
| 티스토리/미디엄 | 외부 플랫폼 의존, 브랜드 통합 불가 |
| 노션 공유 | 검색 엔진 미노출, 구조화 불가 |
| 별도 기술 블로그 구축 | 회사 사이트와 분리, 운영 이중화 |

### 2.3 시장 규모

- 국내 AI 스타트업: 약 3,000개 이상 (2026년 기준, 중기부 통계 참조)
- 기술 조직을 보유한 IT 기업: 약 15,000개 이상
- **잠재 고객**: 콘텐츠 마케팅 필요성을 인식하지만 자체 구축 여력이 없는 중소 기술 기업

## 3. 솔루션

### 3.1 BAIKAL Content Platform의 핵심 가치

```
AI 프로젝트 완성
        ↓
  플랫폼에 등록 (5분)
        ↓
  공개 포트폴리오 자동 생성
        ↓
  기술 아티클 발행 → SEO 트래픽 시작
        ↓
  외부 채널(X, 블로그) 자동 수집·통합
        ↓
  시간이 지날수록 강해지는 신뢰 자산
```

### 3.2 4가지 핵심 기능

#### ① AI 프로젝트 포트폴리오 엔진
- 구조화된 프로젝트 등록 (기술 스택, 성과 수치, 마크다운 설명)
- 카테고리·태그 기반 탐색
- 프로젝트 ↔ 콘텐츠 자동 연결

#### ② 기술 콘텐츠 지식베이스 (LAB)
- 아티클 / 노트 / 실험 기록 / 연구 기록 타입별 관리
- 마크다운 기반 풍부한 콘텐츠 작성
- 코드 하이라이트, 표, 수식 지원

#### ③ 멀티채널 콘텐츠 자동 수집·통합
- RSS, X(Twitter), YouTube, Instagram 자동 수집
- SHA256 해시 기반 중복 제거
- 관리자 검수 워크플로우 (pending → published)
- 수집 주기 설정, 오류 모니터링

#### ④ SEO 자산 누적 엔진
- 프로젝트·콘텐츠별 개별 meta title/description
- Open Graph 이미지 자동 생성
- Schema.org 구조화 데이터
- XML 사이트맵 자동 갱신
- 연도·월별 아카이브

### 3.3 기술 스택 (검증된 조합)

| 영역 | 기술 | 선택 이유 |
|------|------|-----------|
| Frontend | React 18 + TypeScript | 타입 안전성, 생태계 |
| 스타일 | TailwindCSS | 빠른 커스터마이징 |
| DB/Auth | Supabase (PostgreSQL) | 서버리스, 즉시 확장 |
| 벡터 검색 | pgvector | 콘텐츠 시맨틱 검색 확장 가능 |
| 배포 | Vercel + Supabase Cloud | 제로 운영 오버헤드 |
| AI | OpenAI API | 콘텐츠 자동 요약·분류 확장 |

## 4. 비즈니스 모델

### 4.1 단기: 바이칼시스템즈 자체 운영

- 플랫폼을 통한 AI MVP 포트폴리오 공개
- SEO 트래픽 → 인바운드 영업 기회 창출
- 기술 신뢰도 제고 → 채용 경쟁력 강화

### 4.2 중기: 화이트라벨 서비스화

**대상**: AI 스타트업, AI 연구소, 기술 중소기업

**제공 형태**:

| 모델 | 설명 | 예상 단가 |
|------|------|----------|
| 화이트라벨 구축 | 커스터마이징 전달 | 프로젝트당 1,500~3,000만원 |
| SaaS 월정액 | 호스팅 포함 운영 | 월 50~200만원 |
| 유지보수 계약 | 기능 추가·운영 지원 | 월 30~100만원 |

### 4.3 장기: 플랫폼 네트워크 효과

- 다수 AI 기업이 동일 플랫폼 사용
- 콘텐츠 간 연결망(태그·토픽) → AI 생태계 지식 허브
- 플랫폼 내 유료 광고·협업 매칭 서비스
