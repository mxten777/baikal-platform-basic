# BAIKAL AI Content Platform
## 사용자 매뉴얼

> **대상**: 공개 사이트 방문자  
> **사이트**: https://www.baikalsys.kr  
> **작성일**: 2026-03-12  
> **버전**: v1.0

---

## 1. 사이트 소개

BAIKAL AI Content Platform은 **바이칼시스템즈**의 AI 개발 과정과 기술 인사이트를 공유하는 플랫폼입니다.

- AI MVP 프로젝트 포트폴리오
- 기술 아티클, 개발 일지, 연구 기록
- 멀티채널(X, 블로그, YouTube 등) 콘텐츠 큐레이션

## 3. 프로젝트 (`/projects`)

### 3.1 목록 페이지

#### 카테고리 필터

상단 필터 탭으로 카테고리별 프로젝트 조회:

| 탭 | 내용 |
|----|------|
| 전체 | 모든 카테고리 |
| NLP / 언어 | 자연어 처리 프로젝트 |
| AI 에이전트 | 멀티에이전트, 자동화 프로젝트 |
| 플랫폼 / 인프라 | 개발 도구, 인프라 프로젝트 |
| 리서치 / 실험 | 벤치마크, 실험 기록 |
| 컴퓨터 비전 | 이미지·영상 AI 프로젝트 |

#### 프로젝트 카드

각 카드에 표시되는 정보:
- 카테고리 배지
- 프로젝트 제목 및 한 줄 설명
- 주요 기술 스택 태그 (최대 4개)
- 클릭 시 상세 페이지 이동

### 3.2 프로젝트 상세 페이지 (`/projects/:slug`)

| 섹션 | 내용 |
|------|------|
| 헤더 | 제목, 카테고리, 상태 배지 |
| 설명 | 마크다운 본문 (개요, 주요 기능, 성과) |
| 기술 스택 | 사용 기술 목록 |
| 링크 | 데모, 레포지토리 (등록된 경우) |
| 관련 콘텐츠 | 이 프로젝트와 연결된 기술 글 목록 |

## 5. 콘텐츠 허브 (`/content`)

멀티채널에서 수집된 모든 콘텐츠를 한곳에서 볼 수 있는 통합 피드입니다.

### 5.1 콘텐츠 타입 필터

상단 탭에서 타입별 필터:

| 탭 | 포함 콘텐츠 |
|----|------------|
| 전체 | 모든 타입 |
| Article | 기술 블로그 글 |
| Note | 짧은 메모·일지 |
| X Post | X(Twitter) 포스팅 |
| Research | 실험·연구 기록 |

### 5.2 콘텐츠 카드

- **제목**: 콘텐츠 제목
- **요약**: 한 줄 설명 또는 본문 앞부분
- **타입 배지**: article / note / x_post / research
- **게시 날짜**
- **태그**: 해당 글의 기술 키워드

### 5.3 채널별 페이지 (`/content/:channel`)

| 경로 | 채널 |
|------|------|
| `/content/blog` | 블로그 글 |
| `/content/x` | X(Twitter) 큐레이션 |
| `/content/youtube` | YouTube 영상 |
| `/content/instagram` | Instagram 큐레이션 |

## 10. 자주 묻는 질문 (FAQ)

**Q. 콘텐츠를 구독하려면 어떻게 하나요?**  
A. RSS 피드 (`/api/rss.xml`)를 RSS 리더에 등록하면 새 글 알림을 받을 수 있습니다.

**Q. 특정 기술 키워드로 검색하려면?**  
A. 콘텐츠 허브(`/content`)에서 태그 필터를 활용하거나, 토픽 페이지(`/topics`)에서 해당 기술 주제를 찾아보세요.

**Q. 프로젝트 데모를 체험하려면?**  
A. 프로젝트 상세 페이지에서 "데모 보기" 링크가 제공된 경우 클릭하여 접근할 수 있습니다. (프로젝트마다 상이)

**Q. 기술 문의나 프로젝트 의뢰는 어디서 하나요?**  
A. `/contact` 페이지를 이용하거나, 각 페이지 하단 푸터의 연락처를 이용해 주세요.

**Q. 관리자 페이지는 어디에 있나요?**  
A. `/admin`은 인증된 관리자만 접근 가능하며, 일반 사용자는 이용할 수 없습니다.
